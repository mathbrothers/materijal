Ispitivanje funkcije

Postavka

Data je funkcija

\[ f:(-1,+\infty)\to\mathbb{R}, \qquad f(x)=\ln(x^3+1). \]

Domen je

\[ x^3+1>0 \quad\Longleftrightarrow\quad x>-1. \]

Dakle:

\[ D_f=(-1,+\infty). \]

(a) Granične vrednosti

Za \(x\to -1^+\) imamo:

\[ x^3+1\to 0^+. \]

Zato je:

\[ \lim_{x\to -1^+}\ln(x^3+1)=-\infty. \]

Dakle:

\[ \boxed{ \lim_{x\to -1^+} f(x)=-\infty } \]

Za \(x\to+\infty\) imamo:

\[ x^3+1\to+\infty. \]

Zato je:

\[ \boxed{ \lim_{x\to+\infty} f(x)=+\infty } \]

Pošto je

\[ \lim_{x\to -1^+}f(x)=-\infty, \]

prava

\[ \boxed{x=-1} \]

je vertikalna asimptota.

(b) Izvodi

Prvi izvod:

\[ f'(x) = \frac{1}{x^3+1}\cdot 3x^2 = \frac{3x^2}{x^3+1}. \] \[ \boxed{ f'(x)=\frac{3x^2}{x^3+1} } \]

Drugi izvod:

\[ \begin{aligned} f''(x) &= \left(\frac{3x^2}{x^3+1}\right)' \\[6pt] &= \frac{6x(x^3+1)-3x^2\cdot 3x^2}{(x^3+1)^2} \\[6pt] &= \frac{6x^4+6x-9x^4}{(x^3+1)^2} \\[6pt] &= \frac{6x-3x^4}{(x^3+1)^2} \\[6pt] &= \frac{3x(2-x^3)}{(x^3+1)^2}. \end{aligned} \] \[ \boxed{ f''(x)=\frac{3x(2-x^3)}{(x^3+1)^2} } \]

Treći izvod:

\[ \boxed{ f'''(x)= \frac{6(x^6-7x^3+1)}{(x^3+1)^3} } \]

(c) Lokalni ekstremi i prevojne tačke

Lokalni ekstremi

Kritične tačke tražimo iz uslova:

\[ f'(x)=0. \]

Kako je

\[ f'(x)=\frac{3x^2}{x^3+1}, \]

a na domenu je \(x^3+1>0\), dobijamo:

\[ 3x^2=0 \quad\Longleftrightarrow\quad x=0. \]

Međutim, za svako \(x\in(-1,+\infty)\) važi:

\[ f'(x)\geq 0, \]

i \(f'(x)=0\) samo za \(x=0\). Znak izvoda se u \(x=0\) ne menja.

\(x\) \((-1,0)\) \(0\) \((0,+\infty)\)
\(f'(x)\) \(+\) \(0\) \(+\)

Zato funkcija nema lokalni maksimum ni lokalni minimum.

Odgovor:

\[ \boxed{ \text{funkcija nema lokalnih ekstremnih tačaka} } \]

Prevojne tačke

Prevojne tačke tražimo iz uslova:

\[ f''(x)=0. \]

Kako je

\[ f''(x)=\frac{3x(2-x^3)}{(x^3+1)^2}, \]

a imenilac je pozitivan na domenu, rešavamo:

\[ 3x(2-x^3)=0. \] \[ x=0 \quad\text{ili}\quad x=\sqrt[3]{2}. \]

Posmatramo znak drugog izvoda. Znak zavisi od izraza

\[ x(2-x^3). \]
\(x\) \((-1,0)\) \(0\) \((0,\sqrt[3]{2})\) \(\sqrt[3]{2}\) \((\sqrt[3]{2},+\infty)\)
\(f''(x)\) \(-\) \(0\) \(+\) \(0\) \(-\)

Pošto se znak drugog izvoda menja u obe tačke, obe su prevojne tačke.

Za \(x=0\):

\[ f(0)=\ln 1=0. \]

Dakle, prva prevojna tačka je:

\[ \boxed{(0,0)} \]

Za \(x=\sqrt[3]{2}\):

\[ f\left(\sqrt[3]{2}\right) = \ln\left((\sqrt[3]{2})^3+1\right) = \ln(2+1) = \ln 3. \]

Dakle, druga prevojna tačka je:

\[ \boxed{ \left(\sqrt[3]{2},\ln 3\right) } \]

(d) Podaci za skicu grafa

Domen funkcije je:

\[ \boxed{(-1,+\infty)} \]

Vertikalna asimptota je:

\[ \boxed{x=-1} \]

Funkcija je rastuća na celom domenu:

\[ \boxed{ f'(x)\geq 0 \quad\text{za svako }x\in(-1,+\infty) } \]

Tačka \((0,0)\) je stacionarna prevojna tačka, jer je:

\[ f'(0)=0, \qquad f''(0)=0, \]

ali funkcija u toj tački nema ekstrem.

Konkavnost:

\[ \boxed{ f''(x)<0 \quad\text{na}\quad (-1,0)\cup(\sqrt[3]{2},+\infty) } \] \[ \boxed{ f''(x)>0 \quad\text{na}\quad (0,\sqrt[3]{2}) } \]

Prevojne tačke su:

\[ \boxed{(0,0)} \] \[ \boxed{ \left(\sqrt[3]{2},\ln 3\right) } \]

Za skicu je korisno zapamtiti sledeće:

  • graf ima vertikalnu asimptotu \(x=-1\);
  • kad \(x\to -1^+\), graf ide u \(-\infty\);
  • funkcija stalno raste;
  • prolazi kroz tačku \((0,0)\);
  • u tački \((0,0)\) ima horizontalnu tangentu, ali nema ekstrem;
  • menja konkavnost u tačkama \((0,0)\) i \(\left(\sqrt[3]{2},\ln 3\right)\);
  • kad \(x\to+\infty\), važi \(f(x)\to+\infty\).

Konačan sažetak

\[ \boxed{ \lim_{x\to -1^+}f(x)=-\infty } \] \[ \boxed{ \lim_{x\to+\infty}f(x)=+\infty } \] \[ \boxed{ f'(x)=\frac{3x^2}{x^3+1} } \] \[ \boxed{ f''(x)=\frac{3x(2-x^3)}{(x^3+1)^2} } \] \[ \boxed{ f'''(x)= \frac{6(x^6-7x^3+1)}{(x^3+1)^3} } \] \[ \boxed{ \text{nema lokalnih ekstremnih tačaka} } \] \[ \boxed{ (0,0) \quad\text{i}\quad \left(\sqrt[3]{2},\ln 3\right) \text{ su prevojne tačke} } \]
