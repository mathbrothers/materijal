Naći ostatak pri deljenju polinoma \( P(x) = x^{2025} + x^{2024} - x^{2023} + 1 \) polinomom \( Q(x) = x^3 + x^2 + x + 1 \).
Korak 1: Rastavljanje delioca Polinom \( Q(x) \) možemo faktorisati grupisanjem članova: \[ Q(x) = x^2(x+1) + 1(x+1) = (x+1)(x^2+1) \] Nule delioca su \( x = -1 \) i \( x = \pm i \). Ostatak \( R(x) \) mora biti oblika \( ax^2 + bx + c \).
Korak 2: Korišćenje nule x = -1 Vrednost polinoma \( P(x) \) u tački \( -1 \) mora biti jednaka vrednosti ostatka \( R(x) \): \[ P(-1) = (-1)^{2025} + (-1)^{2024} - (-1)^{2023} + 1 = -1 + 1 - (-1) + 1 = 2 \] \[ R(-1) = a(-1)^2 + b(-1) + c = a - b + c \] Dobijamo prvu jednačinu: \( a - b + c = 2 \).
Korak 3: Korišćenje nule x = i Iz \( P(i) = R(i) \) dobijamo: \[ P(i) = i^{2025} + i^{2024} - i^{2023} + 1 = i + 1 - (-i) + 1 = 2i + 2 \] \[ R(i) = a(i)^2 + b(i) + c = -a + bi + c = (c - a) + bi \] Izjednačavanjem realnih i imaginarnih delova dobijamo: \( b = 2 \) i \( c - a = 2 \).
Korak 4: Određivanje koeficijenata Zamenom \( b = 2 \) i \( c = a + 2 \) u prvu jednačinu: \[ a - 2 + a + 2 = 2 \implies 2a = 2 \implies a = 1 \] Vrednost za \( c \) je \( c = 1 + 2 = 3 \).
Ostatak je: \( R(x) = x^2 + 2x + 3 \)
