Logička šema za \( \sqrt{P} > Q \):
\( (P \ge 0 \land Q < 0) \cup (Q \ge 0 \land P > Q^2) \)
Korak 1: Prvi deo formule (P ≥ 0 i Q < 0) Uslov \( P \ge 0 \) predstavlja oblast definisanosti korena: \( 625 - x^2 \ge 0 \), što daje interval \( x \in [-25, 25] \).
Uslov \( Q < 0 \) se odnosi na desnu stranu nejednačine: \( -3x < 0 \), što daje \( x > 0 \).
Presek ova dva uslova je interval (0, 25], što obuhvata 25 celobrojnih rešenja.
Korak 2: Drugi deo formule (Q ≥ 0 i P > Q²) Uslov \( Q \ge 0 \) daje \( -3x \ge 0 \), odnosno \( x \le 0 \).
Uslov \( P > Q^2 \) dobijamo kvadriranjem: \( 16(625 - x^2) > (-3x)^2 \), što se svodi na \( 25x^2 < 10000 \), odnosno \( x \in (-20, 20) \).
Presek uslova \( x \le 0 \) i intervala \( (-20, 20) \) daje interval (-20, 0], što obuhvata 20 celobrojnih rešenja.
Korak 3: Konačan broj rešenja Spajanjem (unijom) rešenja iz oba slučaja dobijamo konačan skup rešenja: \[ x \in (-20, 25] \] Ukupan broj celobrojnih rešenja je \( 20 + 25 = 45 \).
Tačan odgovor: D) 45
