Zadatak: Odrediti vrednost realnog parametra a za koju funkcije f(x) = ax2 - ax + a - 1 i g(x) = 3ax2 - 2x + a imaju jednake (konačne) maksimalne vrednosti.

1. Uslov za postojanje maksimuma

Kvadratna funkcija oblika h(x) = Ax2 + Bx + C ima maksimum samo ako je vodeći koeficijent negativan (A < 0).

  • Za funkciju f(x): a < 0
  • Za funkciju g(x): 3a < 0 ⇒ a < 0

2. Formula za maksimalnu vrednost

Maksimalna vrednost (vrh parabole) se računa po formuli:

ymax =
4AC - B24A

Za funkciju f(x): A=a, B=-a, C=a-1

yf =
4a(a - 1) - (-a)24a
=
3a2 - 4a4a
= (3a - 4) / 4

Za funkciju g(x): A=3a, B=-2, C=a

yg =
4(3a)(a) - (-2)24(3a)
=
12a2 - 412a
= (3a2 - 1) / 3a

3. Izjednačavanje vrednosti

Postavljamo jednačinu yf = yg:

(3a - 4) / 4 = (3a2 - 1) / 3a

Množenjem sa 12a dobijamo:

3a(3a - 4) = 4(3a2 - 1)
9a2 - 12a = 12a2 - 4
3a2 + 12a - 4 = 0

4. Rešavanje jednačine

Rešenja kvadratne jednačine su:

a1,2 = -2 ± (4 / √3)

S obzirom na početni uslov da je a < 0, i da vrednost mora da obezbedi maksimum (teme okrenuto nadole), proveravamo rešenja:

  • a1 = -2 + 4/√3 ≈ 0.31 (Pozitivno, daje minimum - ne odgovara)
  • a2 = -2 - 4/√3 (Negativno - odgovara)
Tačan odgovor: B) -2 - 4/√3
