Zadatak: Odrediti vrednost realnog parametra a za koju funkcije f(x) = ax2 - ax + a - 1 i g(x) = 3ax2 - 2x + a imaju jednake (konačne) maksimalne vrednosti.

1. Uslov za postojanje maksimuma

Da bi kvadratne funkcije imale maksimum (parabole okrenute nadole), vodeći koeficijenti moraju biti negativni:

  • Za f(x): a < 0
  • Za g(x): 3a < 0 ⇒ a < 0

2. Određivanje maksimuma preko prvog izvoda

Maksimum se dostiže u tački gde je prvi izvod jednak nuli (y' = 0).

A) Za funkciju f(x) = ax2 - ax + a - 1:
f'(x) = 2ax - a
2ax - a = 0 ⇒ a(2x - 1) = 0 ⇒ x = 1/2

Vrednost funkcije u toj tački:

f(1/2) = a(1/2)2 - a(1/2) + a - 1 = a4 - a2 + a - 1 = (3a - 4) / 4
B) Za funkciju g(x) = 3ax2 - 2x + a:
g'(x) = 6ax - 2
6ax - 2 = 0 ⇒ 6ax = 2 ⇒ x = 1/3a

Vrednost funkcije u toj tački:

g(1/3a) = 3a(1/3a)2 - 2(1/3a) + a = 13a - 23a + a = (3a2 - 1) / 3a

3. Izjednačavanje i rešavanje

Postavljamo uslov fmax = gmax:

3a - 44 = 3a2 - 13a

Nakon množenja sa 12a i sređivanja:

3a(3a - 4) = 4(3a2 - 1)
9a2 - 12a = 12a2 - 4
3a2 + 12a - 4 = 0

4. Finalni rezultat

Rešavanjem kvadratne jednačine dobijamo:

a1,2 = -2 ± 4√3

S obzirom na uslov a < 0, jedino logično rešenje koje obezbeđuje maksimum (konkavnost nadole) je negativna vrednost:

Tačan odgovor: B) -2 - 4/√3
