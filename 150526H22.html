Taylorov polinom i ostatak

Postavka

Data je funkcija

\[ f:\mathbb{R}\to\mathbb{R}, \qquad f(x)=x^2e^{-x}. \]

Treba odrediti Taylorov polinom trećeg stepena oko tačke

\[ a=-1. \]

Zato će se Taylorov polinom pisati po stepenima izraza

\[ x-a=x+1. \]

(a) Taylorov polinom trećeg stepena

Računamo izvode funkcije:

\[ f(x)=x^2e^{-x} \] \[ f'(x)=e^{-x}(2x-x^2) \] \[ f''(x)=e^{-x}(x^2-4x+2) \] \[ f'''(x)=e^{-x}(-x^2+6x-6) \]

Vrednosti izvoda u tački \(x=-1\) su:

\[ f(-1)=e \] \[ f'(-1)=-3e \] \[ f''(-1)=7e \] \[ f'''(-1)=-13e \]

Taylorov polinom trećeg stepena je:

\[ T_3(f,x,-1) = f(-1) + f'(-1)(x+1) + \frac{f''(-1)}{2!}(x+1)^2 + \frac{f'''(-1)}{3!}(x+1)^3. \]

Ubacivanjem vrednosti dobijamo:

\[ \begin{aligned} T_3(f,x,-1) &= e-3e(x+1)+\frac{7e}{2}(x+1)^2-\frac{13e}{6}(x+1)^3 \\[6pt] &= e\left( 1-3(x+1)+\frac{7}{2}(x+1)^2-\frac{13}{6}(x+1)^3 \right). \end{aligned} \]

Odgovor:

\[ \boxed{ T_3(f,x,-1) = e\left( 1-3(x+1)+\frac{7}{2}(x+1)^2-\frac{13}{6}(x+1)^3 \right) } \]

(b) Ostatak

Po definiciji, ostatak je razlika funkcije i njenog Taylorovog polinoma:

\[ R_3(f,x,-1) = f(x)-T_3(f,x,-1). \]

Zato je:

\[ \boxed{ R_3(f,x,-1) = x^2e^{-x} - e\left( 1-3(x+1)+\frac{7}{2}(x+1)^2-\frac{13}{6}(x+1)^3 \right) } \]
Ako se traži oblik ostatka pogodan za procenu greške, koristi se Lagrangeov oblik ostatka.

Za Lagrangeov ostatak potreban nam je četvrti izvod:

\[ f^{(4)}(x)=e^{-x}(x^2-8x+12). \]

Lagrangeov oblik ostatka glasi:

\[ R_3(f,x,-1) = \frac{f^{(4)}(\xi)}{4!}(x+1)^4, \]

gde je \(\xi\) između \(-1\) i \(x\). Dakle:

\[ \boxed{ R_3(f,x,-1) = \frac{e^{-\xi}(\xi^2-8\xi+12)}{24}(x+1)^4 } \]

pri čemu je \(\xi\) između \(-1\) i \(x\).

(c) Procena ostatka

Treba naći konstantu \(C>0\) takvu da važi:

\[ |f(x)-T_3(f,x,-1)| \leq C|x+1|^4 \]

za svako

\[ x\in[-2,0]. \]

Po Lagrangeovom ostatku imamo:

\[ |R_3(f,x,-1)| = \left| \frac{f^{(4)}(\xi)}{24}(x+1)^4 \right|. \]

Pošto je \(\xi\) između \(-1\) i \(x\), a \(x\in[-2,0]\), dovoljno je posmatrati interval

\[ \xi\in[-2,0]. \]

Na tom intervalu je:

\[ f^{(4)}(\xi) = e^{-\xi}(\xi^2-8\xi+12). \]

Treba nam gornja ocena izraza

\[ e^{-\xi}(\xi^2-8\xi+12) \]

na intervalu \([-2,0]\). Maksimum se dobija u tački \(\xi=-2\), pa je:

\[ f^{(4)}(-2) = e^2(4+16+12) = 32e^2. \]

Zato možemo uzeti:

\[ C=\frac{32e^2}{24}. \]

Odnosno:

\[ C=\frac{4e^2}{3}. \]

Odgovor:

\[ \boxed{ C=\frac{4e^2}{3} } \]

Dakle, za svako \(x\in[-2,0]\) važi:

\[ \boxed{ |f(x)-T_3(f,x,-1)| \leq \frac{4e^2}{3}|x+1|^4 } \]
